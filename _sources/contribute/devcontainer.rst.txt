######################
VS Code - DevContainer
######################

The easiest way to get your development environment setup is by using VS Code
with Dev Containers, this utilises Docker containers to setup a development
environment that guarantees a match with all other developers, removing any
potential headaches from incompatibilities.

************
Requirements
************

* VS Code
* Docker
* Remote Development (VS Code extension pack)


*****
Usage
*****

#. Open VS Code
#. Open the command pallete and type: ``Dev Continers: Clone repository in container volume``

This will then clone the repository directly into the development container and install
all the recommended plugins. You can then start development without any further configuration
