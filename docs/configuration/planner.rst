.. _planneroptions:

###############
Planner Options
###############

These options are specific to the Planner View.

========================= ========= =============== ==========================================================================================================================================================================================================================
 Name                      Type      Default         Description
========================= ========= =============== ==========================================================================================================================================================================================================================
 plannerDaysToShow         integer   optional        ``7`` Number of days to show in the planner view.
 plannerRollingWeek        boolean   optional        ``false`` If set to true, the planner will always start from the current day. If false, it starts from the first day of the week.
========================= ========= =============== ==========================================================================================================================================================================================================================

.. note::
   We recommend using the Planner View in a **Panel** (1-column) view for the best experience, as it requires significant horizontal space to display multiple calendars side-by-side.
